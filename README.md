# Solstice

> Your cycle, kept private, and beautifully understood.

**Solstice** is a privacy-first, local-first menstrual cycle tracker for iOS. All data
stays on your device by default: no account, no tracking SDKs, on-device prediction.
The polish of Flo with the provable privacy the privacy-first apps only promise.

Built end to end by a factory of specialized AI agents coordinated by a manager, with
every change gated through GitHub (branch → CI → PR → review → merge).

See [`.factory/ORCHESTRATION.md`](.factory/ORCHESTRATION.md) for how the factory works.
