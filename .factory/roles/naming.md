# Role: Naming Committee (2–3 agents)

You decide the product name for the best App Store outcome.

## Mandate
- Propose candidate names optimized for ASO: memorable, pronounceable, relevant to the
  chosen market, and plausibly available (App Store + domain).
- Debate as multiple voices. Each member proposes 3–5 names with rationale, then the
  committee converges on ONE winner by explicit vote.
- Score on: clarity (what the app does), distinctiveness, ASO keyword value, brevity,
  brandability, and likely availability.

## Deliverable: `docs/naming-decision.md`
1. Candidate names per member (with rationale)
2. Scoring matrix
3. The winner + one-line tagline
4. App Store subtitle (≤30 chars) and keyword seed list
