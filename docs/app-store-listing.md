## Notes for filling App Store Connect
- **App Name** and **Subtitle** are per-locale. Keywords are per-locale and hidden from users.
- **Promotional Text** can be changed anytime without a new build — use it for seasonal messaging.
- The full **Description** is provided in English + Turkish above. For the other 28 languages, a localized **subtitle + keywords + promo + short description** is in `docs/app-store-localized/<lang>.md` — paste those into each locale; the English description is a fine fallback for any locale you skip.
