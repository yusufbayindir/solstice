# Thai (th)
- **App Name** (24): Solstice: ติดตามรอบเดือน
- **Subtitle** (27): ติดตามประจำเดือนเป็นส่วนตัว
- **Keywords** (99): ประจำเดือน,รอบเดือน,ติดตาม,ตกไข่,เจริญพันธุ์,ความเป็นส่วนตัว,ปฏิทิน,สุขภาพ,offline,pms,ไม่ต้องบัญชี
- **Promotional Text** (128): รอบเดือนของคุณเป็นส่วนตัว Solstice ทำนายประจำเดือนและช่วงเจริญพันธุ์บน iPhone ของคุณ — ไม่ต้องมีบัญชี ไม่มีคลาวด์ ไม่มีการติดตาม
- **Description**:
Solstice คือแอปติดตามประจำเดือนและรอบเดือนที่เก็บข้อมูลส่วนตัวที่สุดของคุณไว้บน iPhone — ไม่ต้องมีบัญชี ไม่มีคลาวด์ ไม่มีการติดตาม แอปทำนายประจำเดือน ช่วงเจริญพันธุ์ และการตกไข่ทั้งหมดบนเครื่องของคุณ

บันทึกประจำเดือน อาการ และอารมณ์ ดูวงแหวนรอบเดือนที่ชัดเจนและปฏิทินที่อ่านง่าย และปลดล็อกข้อมูลเชิงลึก การซิงค์กับ Apple Health และการส่งออกด้วย Solstice+ แบบเลือกใช้

การล็อกด้วย Face ID และการลบในแตะเดียวทำให้รอบเดือนเป็นของคุณ

รอบเดือนของคุณ โทรศัพท์ของคุณ ความเป็นส่วนตัวของคุณ
